مشکل: Port 3000 in use
حل: Stop-Process -Name node -Force

مشکل: MongoDB connection failed
حل: net start MongoDB

مشکل: Redis connection refused
حل: redis-cli ping (بررسی running بودن)
     docker run -d -p 6379:6379 redis

مشکل: curl not found
حل: استفاده از Git Bash یا نصب curl for Windows
