# Screenshots Directory

این پوشه برای ذخیره اسکرین‌شات‌های پروژه است.

## فایل‌های مورد نیاز:

1. `01-login-register.png` - صفحه ورود/ثبت‌نام
2. `02-main-dashboard.png` - Dashboard اصلی
3. `03-add-todo-modal.png` - مودال افزودن TODO
4. `04-todos-with-filters.png` - لیست TODO ها با فیلترها
5. `05-mongodb-connected.png` - لاگ اتصال MongoDB
6. `06-postman-api-test.png` - تست API با Postman
7. `07-jwt-token.png` - نمایش JWT Token
8. `08-rate-limiting.png` - خطای Rate Limiting
9. `09-docker-running.png` - Docker در حال اجرا
10. `10-user-profile.png` - پروفایل کاربر

## راهنما:

برای گرفتن اسکرین‌شات‌ها، صفحه زیر را باز کنید:
```
http://localhost:3000/screenshots-guide.html
```

یا فایل `public/screenshots-guide.html` را در مرورگر باز کنید.
